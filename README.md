# OnePath WebApp

Proyecto de landing page para OnePath, una plataforma de viajes compartidos entre estudiantes universitarios. El sitio está construido con HTML, CSS y JavaScript, utilizando un style guide propio (sin utilidades de Tailwind) para componentes como encabezado, hero, estadísticas, razones, pasos, universidades, seguridad, testimonios, CTA y modales de autenticación.
